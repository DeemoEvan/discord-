# 以下展示以及教學如何利用selenium函式庫自動簽到米米警察:

1.首先去安裝firefox瀏覽器

2.接著將整個zip下載下來
![8f5da834-a03b-4be1-8bb2-a6db504c62f4](https://github.com/user-attachments/assets/cde9a18d-c6af-443f-86c9-41b896ec5e01)

3.接壓縮後有一個login.txt，將dc帳號打在第一行，密碼打在第二行

4.進到test.py，根據檔案位置，將driver、firefox_profile、login.txt路徑改成正確的。
注意，三個必須在同一個資料夾目錄下(不能是子資料夾)!

5.確認是否有下載python編譯環境以及selenium、tqdm函式庫

6.執行test.py即可
