# 以下展示以及教學如何利用selenium函式庫自動簽到米米警察:

1.首先去安裝firefox瀏覽器(不用chrome因不能用新版本)

2.接著將整個zip下載下來(若之後運行有問題請自行安裝driver)(link:[https://github.com/mozilla/geckodriver/releases](url))
(windows選擇win64版)

3.接壓縮後有一個login.txt，將dc帳號打在第一行，密碼打在第二行

4.確認是否有下載python編譯環境以及selenium、tqdm函式庫

5.執行test.py即可
